Sobre mi

Hola, mi nombre es Franco y soy Analista de Sistemas de Computación y Desarrollador .NET.

Comencé desarrollando aplicaciones web con HTML, CSS y JavaScript, tiempo más tarde empecé a utilizar librerías como Bootstrap y React. Luego, comencé a incursionar en tecnologías de backend, más precisamente en C# .NET.

Allí inicié a crear aplicaciones de escritorio con Windows Forms, luego pasé a Windows Professional Foundation y finalmente al desarrollo de aplicaciones web con ASP.NET Framework y ASP.NET Core MVC.

Actualmente me encuentro trabajando de forma autónoma como soporte técnico informático. Mi principal objetivo al día de hoy es poder pertenecer a una organización a la cual le pueda brindar todo mi potencial como desarrollador de software y en la cual pueda continuar creciendo.
